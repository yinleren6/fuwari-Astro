## 引言

日常使用 Windows 总会遇到一些"差一点"的情况：想快速连个 ADB、想删一个删不掉的文件、想一键打开某个系统设置页面。本文整理了多年积累的实用小脚本，全是高频场景。

## 强删文件/文件夹（类似 rm -rf）

Windows 有时会提示"文件正在使用"或"需要管理员权限"导致删不掉。这个脚本利用 `\\?\` 前缀绕过路径限制：

```batch
@echo off
;删除文件（支持通配符）
DEL /F /A /Q \\?\%1

;删除文件夹
RD /S /Q \\?\%1
```

**参数说明：**

- `/S` — 删除所有子目录
- `/Q` — 安静模式
- `/F` — 强制删除只读文件
- `/A` — 按属性选择文件

> 将文件或文件夹拖到这个脚本上即可使用。

## 导出进程列表（dump）

```batch
echo 导出资源使用情况

tasklist /svc>C:\Users\zelda\Desktop\list_svc.txt
```

## QQ 缓存过大问题 — Group2 文件夹

QQ 的 `Group2` 文件夹缓存越来越大，但又不能直接删。通过目录链接把 `Group` 指向 `Group2`，QQ 就能复用已有缓存：

```batch
mklink /d Group .\Group2
```

> 以管理员身份在 QQ 的 image 目录下执行。

## 非 ASCII 字符正则

```regex
[^\x00-\x7f]
```

`\x00-\x7f` 是 ASCII 范围，取反即匹配所有非 ASCII 字符（中文、日文等）。

## cmd 默认代码页乱码修复

```reg
[HKEY_CURRENT_USER\Console\%SystemRoot%_system32_cmd.exe]
"CodePage"=dword:0000fde9  ; 65001 = UTF-8
```

## 获取本机 IP 与公网 IP

```batch
@echo off
chcp 65001>nul
for /f "tokens=2 delims=:" %%i in ('ipconfig ^| find /i "IPv"') do set ip=%%i
echo 本机ip: %ip%

curl https://myip.ipip.net/
```

## 脚本中的 %errorlevel% 判断

```batch
command && echo 成功 || echo 失败
;等同于
command
if "%errorlevel%"=="0" (echo 成功) else echo 失败
```

## 小结

这些小脚本虽然简单，但每个都解决过真实痛点。建议把常用的（IP 查询、ms-settings）做成 `.cmd` 文件放在桌面或快速启动目录，随用随点。
