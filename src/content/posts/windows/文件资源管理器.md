## 引言

文件资源管理器是 Windows 中使用频率最高的工具之一。本文汇集了大量实用技巧，从符号链接到镜像劫持，从隐藏系统文件夹到控制面板速查，帮你把 Explorer 调教得得心应手。

## 一、符号链接、硬链接与目录链接

`mklink` 是 Windows 上一个非常强大的工具，可以创建各种"链接"文件，解决磁盘空间不足、目录迁移等问题。

### 命令格式

```batch
MKLINK [[/D] | [/H] | [/J]] Link Target
```

### 四种链接类型对比

| 类型 | 参数 | 适用对象 | 跨分区 | 特点 |
|------|------|---------|--------|------|
| 文件符号链接 |（无） | 文件 | ✅ | 类似快捷方式，但对应用透明 |
| 目录符号链接 | `/D` | 目录 | ✅ | 创建 `.SYMLINKD`，移动后仍保持引用 |
| 文件硬链接 | `/H` | 文件 | ❌ | 同一文件的多个名字，删除一个不影响 |
| 目录链接 | `/J` | 目录 | 仅本地 | 创建 `.JUNCTION`，注意移动会搬走源文件 |

### 实战场景

**把 Steam 游戏目录映射到另一个盘：**

```batch
mklink /D "E:\epic\GTAV" "E:\steam\steamapps\common\Grand Theft Auto V"
```

**把 Android Studio 项目链接到桌面：**

```batch
mklink /D "C:\Users\zelda\Desktop\AndroidStudio" "E:\code\AndroidStudio"
```

**把 GTA 自定义电台音乐链接过来：**

```batch
mklink /D "C:\Users\zelda\Documents\Rockstar Games\GTA V\User Music" "D:\media\music\GTAV User Music Link"
```

### 注意事项

1. **硬链接上限：** Windows 下 1024 次，Linux 下 65535 次
2. **删除链接：** 像删除普通文件一样删除链接本身，不会影响源文件
3. **管理员权限：** 创建符号链接需要管理员权限（硬链接 `/H` 不需要）
4. **推荐工具：** [Link Shell Extension](https://schinagl.priv.at/nt/hardlinkshellext/hardlinkshellext.html) — 图形化操作各类链接

## 二、映像劫持（Image File Execution Options）

映像劫持是 Windows 的调试功能，可以在一个程序启动时拦截并执行其他命令。利用得当，它能做很多有趣的事。

### 注册表位置

```
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\<程序名.exe>
```

### 1. 劫持 Chrome 的 software_reporter_tool

Chrome 每次更新后都会运行 `software_reporter_tool.exe`，扫个盘，占 CPU。劫持掉它：

```reg
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\software_reporter_tool.exe]
"Debugger"="cmd.exe /c echo Chrome 启动 software_reporter_tool.exe 了&pause"
```

### 2. 记录命令行调用参数

不知道哪个程序在偷偷调用 cmd？劫持它记录下来：

```reg
;劫持 cmd.exe，记录参数到桌面日志
[HKEY_LOCAL_MACHINE\...\Image File Execution Options\cmd.exe]
"Debugger"="cmd.exe /c echo %*>>C:\\Users\\misaka\\Desktop\\invokecmd.log&pause"

;劫持 powershell.exe
[HKEY_LOCAL_MACHINE\...\Image File Execution Options\powershell.exe]
"Debugger"="cmd.exe /c echo %*>>C:\\Users\\misaka\\Desktop\\invokeps.log&pause"
```

### 3. 添加/删除劫持的 CMD 命令

```batch
;添加
reg add "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\软件名.exe" /v "Debugger" /d "cmd /c echo 启动 %date% %time%>>log.txt" /t REG_SZ /f

;删除
reg delete "HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\软件名.exe" /f
```

### 4. 右键菜单添加映像劫持

在 `.exe` 右键菜单添加"调试"子菜单，包含开启/关闭/自定义劫持：

**添加：**

```reg
[HKEY_CLASSES_ROOT\exefile\shell\debugger]
"MUIVerb"="调试"
"SubCommands"="Ondebugger;Nodebugger;Zdydebugger"
"icon"="%systemroot%\\system32\\imageres.dll,102"

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\CommandStore\shell\Ondebugger]
@="开启劫持(&Y)"
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\CommandStore\shell\Ondebugger\command]
@="cmd /c for /f \"delims=\" %%t in ('dir /b \"%1\"') do reg add \"HKLM\\SOFTWARE\\Microsoft\\Windows NT\\CurrentVersion\\Image File Execution Options\\%%t\\\" /f /v \"debugger\" /t REG_SZ /d \"\\\\\"\""
```

**删除：**

```reg
[-HKEY_CLASSES_ROOT\exefile\shell\debugger]
[-HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\CommandStore\shell\Ondebugger]
[-HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\CommandStore\shell\Zdydebugger]
[-HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\CommandStore\shell\Nodebugger]
```

## 三、用映像劫持替换记事本（Notepad3）

用 Notepad3 替代系统记事本，享受更好的编辑体验：

```reg
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Image File Execution Options\notepad.exe]
"Debugger"="\"C:\\Program Files\\Notepad3\\Notepad3.exe\" /z"
```

同时添加右键"Open with Notepad3"：

```reg
[HKEY_CLASSES_ROOT\*\shell\Open with Notepad3]
"Icon"="C:\\Program Files\\Notepad3\\Notepad3.exe,0"
@="Open with Notepad3"

[HKEY_CLASSES_ROOT\*\shell\Open with Notepad3\command]
@="\"C:\\Program Files\\Notepad3\\Notepad3.exe\" \"%1\""
```

## 五、隐藏此电脑左侧导航栏第三方图标

```batch
HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\MyComputer\Namespace
```

第三方软件（爱奇艺、百度网盘、WPS）的图标也多在此处注册，删除对应子项即可。

## 八、任务栏美化

### 模糊透明

```reg
[HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Themes\Personalize]
"EnableTransparency"=dword:00000001
"AppsUseLightTheme"=dword:00000001
"SystemUsesLightTheme"=dword:00000001
```

### 禁用 Explorer 预读 .dat 文件

```reg
[-HKEY_CLASSES_ROOT\.dat\ShellEx\{BB2E617C-0920-11d1-9A0B-00C04FC2D6C1}]
```

## 九、常用操作脚本

**刷新壁纸（不重启 Explorer）：**

```batch
RunDll32 USER32.DLL,UpdatePerUserSystemParameters
```

**重启 Explorer：**

```batch
taskkill /f /im explorer.exe & start explorer
```

**修复桌面图标缓存：**

```batch
del %userprofile%\AppData\Local\IconCache.db /Q /F
taskkill /im explorer.exe /f & explorer
```

## 十一、磁盘管理隐藏 U 盘 EFI 分区

```batch
diskpart
lis dis           ;列出所有磁盘
sel dis 2         ;选择 U 盘
lis par           ;列出所有分区
sel par 2         ;选择 EFI 分区
set id=06         ;取消分区隐藏
remove letter=k   ;移除盘符
set id=16         ;重新隐藏
```

## 十二、自定义 CMD 窗口

```batch
@echo off
@title 窗口标题
color 3f          ;背景色+字体色
cls
MODE con: COLS=77 LINES=36
pause
```

**颜色代码：** 0=黑 1=蓝 2=绿 3=浅蓝 4=红 5=紫 6=黄 7=白 8=灰 9=淡蓝 A=淡绿 B=淡青 C=淡红 D=淡紫 E=淡黄 F=亮白

## 十三、路径检测（PowerShell）

检测指定目录下超过 260 字符路径的文件：

```powershell
$startPath = "F:\mods\l4d2"
$maxLength = 260
Get-ChildItem -Path $startPath -Recurse -File -Force | ForEach-Object {
    $fullPath = $_.FullName
    if ($fullPath.Length -gt $maxLength) {
        [PSCustomObject]@{
            '文件名'   = $_.Name
            '完整路径' = $fullPath
            '路径长度' = $fullPath.Length
        }
    }
} | Out-GridView -Title "路径过长的文件列表"
```

## 修改文件时间

```powershell
# 批量修改
Get-ChildItem -Path 'E:\game\...' -Recurse | foreach-object{
    $_.LastWriteTime="2020/11/13 16:00"
    $_.CreationTime="2020/11/13 16:00"
}
# 单个文件
(Get-Item 'E:\1.txt').LastWriteTime="2020/11/13 16:00"
```
