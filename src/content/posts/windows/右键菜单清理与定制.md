## 引言

右键菜单是 Windows 日常使用的高频交互点。但装过的软件会不断往里面塞菜单项，久而久之变得又长又卡。本文从 Win11 经典菜单切换开始，涵盖清理残留、添加自定义项、恢复被删项的全流程。

## 一、Win11 恢复经典右键菜单

Windows 11 改成了折叠式右键菜单，很多常用功能要多点一次"显示更多选项"。如果你怀念 Win10 的经典菜单，一行注册表搞定：

```reg
[HKEY_CURRENT_USER\Software\Classes\CLSID\{86ca1aa0-34aa-4e8b-a509-50c905bae2a2}\InProcServer32]
@=""
```

重启 Explorer 后生效。想恢复 Win11 样式就删掉这个键。

## 二、文件夹空白处右键菜单：添加 WizTree

WizTree 是一个磁盘空间分析工具，比 WinDirStat 快很多。把它加到右键菜单，随时分析任意文件夹：

```reg
;驱动器背景
[HKEY_CLASSES_ROOT\Drive\shell\WizTree]
@="&WizTree"
"Icon"="\"E:\\PortableApp\\wiztree\\WizTree64.exe\", 0"
[HKEY_CLASSES_ROOT\Drive\shell\WizTree\command]
@="\"E:\\PortableApp\\wiztree\\WizTree64.exe\" \"%1\""

;文件夹
[HKEY_CLASSES_ROOT\Folder\shell\WizTree]
@="&WizTree"
"Icon"="E:\\PortableApp\\wiztree\\WizTree64.exe,0"
[HKEY_CLASSES_ROOT\Folder\shell\WizTree\command]
@="\"E:\\PortableApp\\wiztree\\WizTree64.exe\" \"%1\""

;目录背景
[HKEY_CLASSES_ROOT\Directory\shell\WizTree]
@="&WizTree"
"Icon"="E:\\PortableApp\\wiztree\\WizTree64.exe,0"
[HKEY_CLASSES_ROOT\Directory\shell\WizTree\command]
@="\"E:\\PortableApp\\wiztree\\WizTree64.exe\" \"%1\""

;文件夹空白处
[HKEY_CLASSES_ROOT\Directory\Background\shell\WizTree]
@="&WizTree"
"Icon"="E:\\PortableApp\\wiztree\\WizTree64.exe,0"
[HKEY_CLASSES_ROOT\Directory\Background\shell\WizTree\command]
@="\"E:\\PortableApp\\wiztree\\WizTree64.exe\" \"%1\""
```

> 路径中的 `E:\PortableApp\wiztree\` 替换成你的实际安装路径。

## 三、右键添加"打开方式"菜单

所有文件都加上"Open With"选项，方便手动选择程序：

```reg
[HKEY_CLASSES_ROOT\*\shell\OpenWith]
@="Open With"
[HKEY_CLASSES_ROOT\*\shell\OpenWith\command]
@="\"C:\\Windows\Sysrem32\\rundll32.exe Shell32.dll,OpenAs_RunDLL %1\""
```

## 四、新建文件类型添加到右键菜单

默认右键"新建"里只有文件夹、文本文档等少数类型。可以添加 `.md` 和 `.xmind` 等：

```reg
;新建 .md 文件
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.md]
@="Typora.md"
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.md\ShellNew]
"NullFile"=""
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.md\OpenWithProgids]
"Nutstore.LightApp.md"=""
"Typora.md"=""
"VSCode.md"=""

;新建 .xmind 文件
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.xmind]
@="XMind Workbook"
[HKEY_LOCAL_MACHINE\SOFTWARE\Classes\.xmind\ShellNew]
"NullFile"=""
```

## 五、清理第三方软件右键残留

这是右键菜单管理中最实用的部分。装过又卸载的软件经常留下死掉的菜单项。

### 删除常用软件残留

| 软件 | 注册表路径 |
|------|-----------|
| 百度云管家 | `HKCR\Directory\shellex\ContextMenuHandlers\YunShellExt` |
| 坚果云 | `HKCR\*\shellex\ContextMenuHandlers\NutstoreExt` |
| Visual Studio | `HKCR\Directory\shell\AnyCode` |
| Intel 核显 | `HKCR\Directory\background\shellex\ContextMenuHandlers\igfxDTCM` |
| QQ 发送到 | `HKCR\Folder\shellex\ContextMenuHandlers\QQShellExt` |
| UltraISO | `HKCR\Folder\shellex\ContextMenuHandlers\UltraISO` |
| 格式工厂 | `HKCR\Directory\shellex\ContextMenuHandlers\FormatFactoryShell` |
| 火萤 | `HKCR\Directory\Background\shellex\ContextMenuHandlers\HYWindow` |
| Open with Code | `HKCR\*\shell\VSCode` |

### 屏蔽系统自带菜单项

如果你觉得某些系统功能用不上，可以用 Blocked 列表屏蔽：

```reg
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Shell Extensions\Blocked]

;包含到库中
"{3dad6c5d-2167-4cae-9914-f99e41c12cfa}"="Include In Library"

;恢复以前的版本
"{596AB062-B4D2-4215-9F74-E9109B0A8153}"="Restore Previous Versions"

;共享
"{e2bf9676-5f8f-435c-97eb-11607a5bedf7}"="Share"

;Cast to Device
"{7AD84985-87B4-4a16-BE58-8B72A5B390F7}"="Cast to Device"

;授予访问权限
"{f81e9010-6ea4-11ce-a7ff-00aa003ca9f6}"="Give access to"

;兼容性疑难解答
"{1d27f844-3a1f-4410-85ac-14651078412d}"="Troubleshoot compatibility"
```

### 屏蔽"固定到"菜单

```reg
;Pin to Start
[-HKCR\exefile\shellex\ContextMenuHandlers\PintoStartScreen]
[-HKCR\Folder\shellex\ContextMenuHandlers\PintoStartScreen]

;Pin to Taskbar
[-HKCR\*\shellex\ContextMenuHandlers\{90AA3A4E-1CBA-4233-B8BB-535773D48449}]

;Pin to 快速访问
[-HKCR\Folder\shell\pintohome]
```

## 六、恢复被误删的右键菜单

误删了菜单项怎么办？对应的 CLSID 补回来即可：

```reg
;百度云管家
[HKEY_CLASSES_ROOT\Directory\shellex\ContextMenuHandlers\YunShellExt]
@="{6D85624F-305A-491d-8848-C1927AA0D790}"

;UltraISO
[HKEY_CLASSES_ROOT\Directory\shellex\ContextMenuHandlers\UltraISO]
@="{AD392E40-428C-459F-961E-9B147782D099}"

;格式工厂
[HKEY_CLASSES_ROOT\Directory\shellex\ContextMenuHandlers\FormatFactoryShell]
@="{A3777921-CFD3-4A6B-89BF-08E6B95716E8}"

;坚果云
[HKEY_CLASSES_ROOT\*\shellex\ContextMenuHandlers\NutstoreExt]
@="{5D652B62-B702-496A-92BC-92C308251FEA}"

;Recuva
[HKEY_CLASSES_ROOT\Directory\shellex\ContextMenuHandlers\RecuvaShellExt]
@="{435E5DF5-2510-463C-B223-BDA47006D002}"
```

## 小结

右键菜单管理其实就三件事：**添加**需要的快捷操作，**清理**没用的软件残留，**屏蔽**不用的系统功能。定期打理一下，右键菜单就能保持清爽高效。
