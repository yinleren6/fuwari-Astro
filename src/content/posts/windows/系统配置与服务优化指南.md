## 引言

系统配置是 Windows 优化中最有"技术感"的部分。服务管理、网络共享、远程桌面加速、休眠唤醒排查——这些配置直接影响系统的稳定性和使用体验。本文汇总了多年积累的系统级配置方案。

## 一、服务管理

### 强制修改服务启动类型

大部分服务可以在服务管理器（`services.msc`）直接修改，但有些系统服务按钮是灰色的。这时可以通过注册表强制修改：

```batch
rem 停止服务
net stop appxsvc

rem 修改启动类型为禁用(4)
reg add "HKLM\SYSTEM\ControlSet001\Services\AppXSvc" /v Start /t REG_DWORD /d 4

rem 删除失败后的恢复动作（防止自动重启）
reg delete "HKLM\SYSTEM\ControlSet001\Services\AppXSvc" /v FailureActions
```

### AppX Deployment Service 禁用

```reg
[HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Services\AppXSvc]
"Start"=dword:00000003  ; 3 = 手动
```

### 查询与停止驱动

```batch
;查询驱动状态
sc query sysdiag

;停止驱动（需要管理员权限）
sc stop mhyprot2
```

## 二、网络共享

### 恢复/自定义 IP 段

Windows 热点默认使用 `192.168.137.1`，可以自定义为其他网段：

**恢复默认：**

```reg
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\SharedAccess\Parameters]
"ScopeAddress"="192.168.137.1"
"ScopeAddressBackup"="192.168.137.1"
```

**自定义为 192.168.18.1：**

```reg
"ScopeAddress"="192.168.18.1"
"ScopeAddressBackup"="192.168.18.1"
```

### 网络共享重启不失效

```reg
[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\SharedAccess]
"EnableRebootPersistConnection"=dword:00000001
```

### 修复文件共享功能和 SMB

```reg
;启用 Workstation 服务和 SMB 协议
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\LanmanWorkstation]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\mrxsmb]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\mrxsmb10]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\mrxsmb20]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\browser]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\NetBIOS]
"Start"=dword:00000002

[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\lmhosts]
"Start"=dword:00000002
```

## 三、远程桌面优化

### 硬件加速 + 60FPS

被控端添加注册表值，提高远程桌面帧率：

```reg
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations]
"DWMFRAMEINTERVAL"=dword:0000000f
; 1000/15 ≈ 66fps，值越小帧数越高
```

控制端组策略设置：

1. 运行 `gpedit.msc`
2. 计算机配置 → 管理模板 → Windows 组件 → 远程桌面服务 → 远程桌面会话主机 → 远程会话环境
3. 启用"将硬件图形适配器应用于所有远程桌面服务会话"
4. 启用"配置远程桌面连接的 H.264/AVC 硬件编码"

### 自定义远程桌面端口

```reg
[HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server\WinStations\RDP-Tcp]
"PortNumber"=dword:00000d3d  ; 3389 的十六进制
```

### RemoteFX USB 重定向

组策略中启用：计算机配置 → 管理模板 → Windows 组件 → 远程桌面服务 → 远程桌面会话客户端 → RemoteFX USB 设备重定向

## 四、睡眠 / 休眠 / 唤醒排查

电脑莫名其妙被唤醒？用以下命令排查：

### 常用命令

```batch
;查看上次唤醒设备
powercfg -lastwake

;查看哪些设备可以唤醒电脑
powercfg -devicequery wake_armed

;查看阻止休眠的进程或驱动
powercfg /requests

;查看计划任务唤醒
powershell "Get-ScheduledTask | where {$_.settings.waketorun}"

;查看电源需求重写
powercfg /requestsoverride

;查看支持的睡眠状态
powercfg -a

;生成电池报告
powercfg /batteryreport
```

### 睡眠状态（S 状态）

| 状态 | 说明 | 唤醒速度 |
|------|------|---------|
| S0 | 正常工作 | - |
| S1 | CPU 停止 | ~0 秒 |
| S2 | CPU 关闭 | ~0.1 秒 |
| S3 | 仅内存供电 | ~0.5 秒 |
| S4 | 休眠（写入硬盘） | ~30 秒 |
| S5 | 关机 | - |

### 禁止硬件唤醒

```batch
powercfg /devicedisablewake "HID-compliant mouse (003)"
powercfg /deviceenablewake "HID-compliant mouse (003)"
```

### 禁止计划任务和定时器唤醒

电源选项 → 更改计划设置 → 更改高级电源设置 → 睡眠 → 允许使用唤醒定时器 → 禁用

### powercfg 请求覆盖

当某个进程阻止电脑睡眠时，可以绕过它的请求：

```batch
;阻止 chrome 阻止睡眠
powercfg /requestsoverride PROCESS "chrome.exe" system

;阻止音频驱动阻止睡眠
powercfg /requestsoverride DRIVER "Realtek High Definition Audio" SYSTEM

;恢复（不加电源需求类型即为恢复）
powercfg /requestsoverride PROCESS chrome.exe
```

### 无人超时睡眠设置

默认唤醒后 120 秒会再次睡眠，可修改此值：

```batch
;使用电池
powercfg /SETDCVALUEINDEX SCHEME_CURRENT 238C9FA8-0AAD-41ED-83F4-97BE242C8F20 7bc4a2f9-d8fc-4469-b07b-33eb785aaca0 36000

;插电源
powercfg /SETACVALUEINDEX SCHEME_CURRENT 238C9FA8-0AAD-41ED-83F4-97BE242C8F20 7bc4a2f9-d8fc-4469-b07b-33eb785aaca0 36000
```

## 五、程序组策略关联启动

通过计划任务 + 事件触发器，可以在某个程序启动或退出时自动执行另一个程序。

### 配置步骤

1. **启用审核策略：** `secpol.msc` → 本地策略 → 审核策略 → 审核进程跟踪 → 勾选"成功"
2. **创建计划任务**（`taskschd.msc`）：
   - 勾选"最高权限运行"
   - 触发器：发生事件时 → 自定义 → XML 编辑

### 事件筛选器 XML

**进程启动时（EventID 4688）：**

```xml
<QueryList>
  <Query Id="0" Path="Security">
    <Select Path="Security">
     *[System[Provider[@Name='Microsoft-Windows-Security-Auditing'] and Task = 13312 and (band(Keywords,9007199254740992)) and (EventID=4688)]]
     and
     *[EventData[Data[@Name='NewProcessName'] and (Data='C:\路径\你的软件A.exe')]]
    </Select>
  </Query>
</QueryList>
```

**进程退出时（EventID 4689）：**

```xml
<QueryList>
  <Query Id="0" Path="Security">
    <Select Path="Security">
     *[System[Provider[@Name='Microsoft-Windows-Security-Auditing'] and Task = 13313 and (band(Keywords,9007199254740992)) and (EventID=4689)]]
     and
     *[EventData[Data[@Name='ProcessName'] and (Data='C:\路径\你的软件A.exe')]]
    </Select>
  </Query>
</QueryList>
```

## 六、Win10 无用功能

以下服务可以按需禁用：

- **诊断跟踪：** Diagnostic Policy、Diagnostic Service Host、Diagnostic System Host
- **连接体验：** Connected User Experiences and Telemetry
- **传递优化：** Windows 更新 → 高级选项 → 传递优化 → 允许从其他电脑下载（关闭）

## 七、Windows 11 优化

### 卸载小组件（Windows Web Experience）

```batch
winget uninstall "windows web experience pack"
```

### 移除 Xbox 游戏服务和 Edge 游戏助手

```powershell
;移除游戏服务
get-appxpackage Microsoft.GamingServices | remove-AppxPackage -allusers

;移除 Edge 游戏助手
Get-AppxPackage -AllUsers -Name Microsoft.Edge.GameAssist | Remove-AppxPackage -allusers
```

## 八、应用使用记录（注册表位置）

Windows 会记录你的应用使用历史，以下位置存储了这些数据：

```
HKCU\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer\FeatureUsage\*
HKCU\SOFTWARE\...\AppCompatFlags\Compatibility Assistant\Store
HKCU\SOFTWARE\Classes\Local Settings\...\MuiCache
HKLM\SYSTEM\ControlSet001\Services\bam\State\UserSettings\S-1-5-21-...
```

## 九、BypassNRO（绕过首次登录联网）

重装系统时跳过强制联网登录：

```batch
reg add HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\OOBE /v BypassNRO /t REG_DWORD /d 1 /f
shutdown /r /t 0
```

## 十、重置 Windows Update 缓存

```batch
@echo off
net stop wuauserv
net stop cryptSvc
net stop bits
net stop msiserver
ren C:\Windows\SoftwareDistribution SoftwareDistribution.old
ren C:\Windows\System32\catroot2 catroot2.old
net start wuauserv
net start cryptSvc
net start bits
net start msiserver
```

## 十一、Win10 优化批处理

有一个大型菜单式批处理脚本（`win10优化.cmd`），包含近 50 项优化选项：
- 关闭 Windows Defender
- 关闭防火墙
- 禁止自动更新驱动
- 关闭系统还原
- 关闭 UAC
- 关闭 IPv6
- 关闭家庭组
- 禁用视觉效果
- 转移临时文件夹
- 禁用远程协助
- 等

执行后选择对应数字即可操作。

## 小结

系统配置优化需要根据实际使用场景来权衡。远程桌面用户关注加速和 USB 重定向，笔记本用户关注睡眠唤醒问题，NAS 用户关注 SMB 文件共享。找到你的需求点，精准配置即可。
