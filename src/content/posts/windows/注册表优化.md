## 引言

注册表是 Windows 的"中枢神经"。很多系统行为没有图形界面开关，只能通过注册表修改。本文整理了多年积累的注册表优化项，涵盖系统清理、个性化、性能优化、硬件调整、网络设置等各个方面。

> ⚠️ **修改注册表前请备份！** 建议先导出整个分支再操作。误删可能导致系统异常。

## 一、系统清理

### 1. 清除多余的 DLL 文件

共享 DLL 引用计数为 0 的可以删除：

```
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\ShareDlls
```

括号内数字为共享引用计数，为 0 则说明已无程序使用。

### 2. 安装卸载残留信息

```
HKCU\Software
HKLM\Software
```

### 3. 删除多余时区（只留北京）

```
HKLM\SOFTWARE\Microsoft\Windows NT\CurrentVersion\Time Zones
```

只保留 `China Standard Time`，其余删除。

### 4. 清除国家列表（只留中国）

```
HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Telephony\Country List
```

86 号文件夹是中国，其余可删。

### 5. 清除多余语言代码（只留中英文）

```
HKLM\SYSTEM\CurrentControlSet\Control\Nls\Locale
```

保留 `0409`（英语）和 `0804`（中文）。

### 6. 删除多余的键盘布局

```
HKLM\SYSTEM\CurrentControlSet\Control\Keyboard Layouts
```

## 二、个性化设置

### 1. 去掉桌面快捷方式小箭头

```
HKEY_CLASSES_ROOT\LNKFILE
```

删除 `IsShortcut` 键。

### 2. 隐藏桌面所有图标

```
HKCU\Software\Windows\CurrentVersion\Policies\Explorer
```

新建 `NoDesktop`（REG_DWORD）= 1

### 3. 修改回收站名称及图标

```
HKLM\SOFTWARE\Classes\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}
```

修改 `(默认)` 为新名称，`DefaultIcon` 子项中修改图标路径。

### 4. 去掉桌面网上邻居

```
HKCU\Software\...\Policies\Explorer
```

新建 `NoNetHood`（REG_DWORD）= 1

### 5. 桌面显示系统版本号

```
HKCU\Control Panel\Desktop
```

`PaintDesktopVersion` = 1

### 6. 改变窗口按钮字体颜色

```
HKCU\Control Panel\Colors
```

`ButtonText` = 如 `255 0 0`（红色）

### 7. 屏蔽桌面 IE 图标

```
HKCU\Software\...\Policies\Explorer
```

新建 `NoInternetIcon`（REG_DWORD）= 1

### 8. 禁止更改桌面壁纸

```
HKCU\Software\...\Policies\Explorer
```

新建 `NoChangingWallPaper`（REG_DWORD）= 1

### 9. 设置登录背景

```
HKU\.DEFAULT\Control Panel\Desktop
```

`WallPaper` = 图片完整路径

### 10-14. 更改系统图标提示信息

| 项目 | 注册表位置 |
|------|-----------|
| 我的电脑 | `HKCR\CLSID\{20D04FE0-3AEA-1069-A2D8-08002B30309D}` |
| 回收站 | `HKCR\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}` |
| IE | `HKCR\CLSID\{871C5380-42A0-1069-A2EA-08002B30309D}` |
| 网上邻居 | `HKCR\CLSID\{208D2C60-3AEA-1069-A2D7-08002B30309D}` |
| 我的文档 | `HKCR\CLSID\{450D8FBA-AD25-11D0-98A8-0800361B1103}` |

修改对应 `InfoTip` 键值为你想要显示的文字。

### 15. 更改桌面图标顺序

```
HKLM\SOFTWARE\Classes\CLSID\{450D8FBA-AD25-11D0-98A8-0800361B1103}
```

`SortOrderIndex` = 48（我的文档第一）/ 54（我的电脑第一）

### 16. 禁止帮助提示信息

```
HKCU\Software\...\Explorer\Advanced
```

`showinfotip` = 0

### 17. 真彩色显示图标

```
HKCU\Control Panel\Desktop\WindowMetrics
```

`Shell Icon BPP` = 32

### 18. 更改图标显示尺寸

```
HKCU\Control Panel\Desktop\WindowMetrics
```

`Shell Icon SIZE` = 48（1024×768）/ 64（1600×1200）

### 19-26. 开始菜单设置

以下都在 `HKCU\Software\...\Policies\Explorer`：

| 功能 | 键名 | 值 |
|------|------|----|
| 自动清除文档历史 | `CleanRecentDocsOnExit` | 1 |
| 禁止文档历史 | `NoRecentDocsHistory` | 1 |
| 去掉"文档" | `NoRecentDocsMenu` | 1 |
| 去掉"查找" | `NoFind` | 1 |
| 去掉"运行" | `NoRun` | 1 |
| 禁止更改开始菜单 | `NoChangeStartMenu` | 1 |
| 隐藏 Windows Update | `NoWindowsUpdate` | 1 |
| 屏蔽"关闭计算机" | `NoClose` | 1 |

### 27. 加快菜单显示速度

```
HKCU\Control Panel\Desktop
```

`MenuShowDelay` = 100 或更小（默认 400）

### 28. 回收站右键菜单增加"删除"和"重命名"

```
HKCR\CLSID\{645FF040-5081-101B-9F08-00AA002F954E}\ShellFolder
```

`Attributes` = `70 01 00 20`

### 29-30. 右键菜单添加项目

**在新窗口打开：**

```
HKLM\...\Directory\shell\NewWindow\command
@="explorer.exe"
```

**快速关闭计算机：**

```
HKLM\...\Directory\shell\快速关闭计算机\command
@="c:\windows\rundll32.exe user.exe,exitwindowsexec"
```

### 31-33. 任务栏设置

| 功能 | 路径 | 键值 |
|------|------|------|
| 禁止任务栏右键菜单 | `...Policies\Explorer` | `NoTrayContextMenu` = 1 |
| 修改任务栏时间文字 | `...Control Panel\International` | `sTimeFormat` |
| 禁止使用任务栏 | `...Policies\Explorer` | `NoSetTaskBar` = 1 |

| 屏蔽设置 | `NoDispSettingsPage` = 1 |

### 5. 屏蔽屏幕保护程序密码

```
HKCU\Software\Policies\Microsoft\Control Panel\Desktop

`ScreenSaverIsSecure` = 1

### 6-12. 添加删除程序设置屏蔽

| 功能 | 键名 |
|------|------|

```

### 19. 自动刷新窗口

```
HKLM\SYSTEM\...\Control\Update
```

`UpdateMode` = 0（自动刷新）

### 20-27. 更改目录位置

在 `HKU\.DEFAULT\...\Explorer\Shell Folders` 中修改：

| 键名 | 说明 |
|------|------|
| `Start Menu` | 开始菜单 |
| `Programs` | 程序 |
| `StartUp` | 启动 |
| `SendTo` | 发送到 |
| `History` | 历史记录 |
| `Personal` | 我的文档 |

## 四、硬件设置

### 1. 启用 CPU 二级缓存

```
HKLM\SYSTEM\...\Memory Management
```

`secondleveldatacache` — 根据 CPU 型号设置（如 200）

### 2-3. 修改 CPU 认证标识

```
HKLM\HARDWARE\...\CentralProcessor\0
```

- `ProcessorNameString` — 修改 CPU 名称（娱乐向）
- `VendorIdentifer` — 修改供应商标识

### 4. 清除内存中不使用的 DLL

```
HKLM\SOFTWARE\...\Explorer
```

新建 `Always UnloadDll`（REG_DWORD）= 1

### 5. 启动自动优化磁盘

```
HKLM\SOFTWARE\Microsoft\Dfrg\BootOptimizeFunction
```

`Enable` = Yes

## 五、杂项注册表修改

### 不允许遥测

```reg
[HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows\DataCollection]
"AllowCommercialDataPipeline"=dword:00000000
"AllowTelemetry"=dword:00000000
```

### IE 浏览器主页设为空白页

```reg
[HKEY_CURRENT_USER\Software\Microsoft\Internet Explorer\Main]
"Start Page"="about:blank"
```

### 蓝牙音箱绝对音量

禁用蓝牙绝对音量，解决音量无法调大的问题：

```reg
[HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Bluetooth\Audio\AVRCP\CT]
"DisableAbsoluteVolume"=dword:00000001
```

### 登陆界面默认打开小键盘

```reg
[HKEY_USERS\.DEFAULT\Control Panel\Keyboard]
"InitialKeyboardIndicators"="2147483648"
```

### 修复键盘驱动注册表

```reg
写入 HKLM\SYSTEM\...\Class\{4d36e96b-e325-11ce-bfc1-08002be10318}
下的各子项，包含 PS/2 Keyboard 和 HID Keyboard Device 的驱动信息
```

### 自定义浏览器 URL 协议（clash://）

```reg
[HKEY_CLASSES_ROOT\clash]
"URL Protocol"=""
@="URL:clash"

[HKEY_CLASSES_ROOT\clash\shell\open\command]
@="\"E:\\Program Files\\Clash for Windows\\Clash for Windows.exe\" \"%1\""
```

### 防火墙注册表项

```
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\WindowsFirewall
```

### Windows Defender 禁用

```
HKEY_LOCAL_MACHINE\SOFTWARE\Policies\Microsoft\Windows Defender
```

删除以下两项：
- `DisableAntiSpyware`
- `DisableAntiVirus`

### 网络列表（网络位置类型）

```
HKLM\SOFTWARE\...\NetworkList\Profiles
HKLM\SOFTWARE\...\NetworkList\Signatures\Unmanaged
```

## 小结

注册表优化需要有的放矢。建议按需选择：
- **系统清理：** 重装系统后一次性操作
- **个性化：** 按自己喜好调整
- **系统设置：** 适合追求精简和性能的用户
- **硬件设置：** 真实性存疑，部分优化效果可能微乎其微
- **杂项：** 解决具体问题的实用条目

记住：注册表修改前一定要备份。一项错误的修改可能导致软件无法运行甚至系统不稳定。
